Guide and Information

Function Info Shortcut
Input Textarea: Where you put input HTML -
Extract Value Inside Only:
  • Checked: Remove HTML tag, take inner value
  • Unchecked (Default): Preserve HTML tag
 -
Keep Line Breaks:
  • Checked (Default): Preserve current line break
  • Unchecked: Remove all line break
 -
Keep Indent:
  • Checked (Default): Preserve current indent
  • Unchecked: Remove all indent (eg tab, space)
 -
Input Class: Enter the class name you want to extract. Example <p class="class-ok"></p>, the class is class-ok
Input ID: Enter the class name you want to extract. Example <p id="id-ok"></p>, the id is id-ok
Input Tag: Enter HTML tag you want to extract. Example for code <p>Testing<a href="https://example.com>Website<a></p>, the HTML tag inside it are p,a .To extract multiple class, comma-separated between tag
Format Output:
  • Checked: Using basic HTML Formating
  • Unchecked (Default): Preserve current input formatting
Uncheck and check the checkbox again to change the output formatting. 		-
Output Textarea: Where the process extract will be output -
Extract HTML Element: Extract only selected Class,ID and HTML tag. If the element are nested, will include all inside the main nested. If all input leaved as blank, will passthrough to output
Extract Text Only: This will extract all text and remove all HTML tag (note: will ignore Class, ID and Tags input)
Send to Input: Current output will replace input text area -
